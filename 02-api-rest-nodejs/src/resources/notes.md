 *** Initiating the project   

    **npm init -y -> For creating package.json** 

* Node doesn't understand TypeScript

So, we need install the Typescript like a development dependency

    **- npm i -D typescript**

*** We need create a settings file typescript

    **npx tsc --init**

*** To run the code

    **npx tsc src/index.ts**

*** Let's go using the Fastify

    **npm i fastify**

*** To solve some problems typescript with NodeJS

    **npm install -D @types/node**

*** Install TSX for convert typescript for javascript, processing typescript and don't create a new file JS
    ** TSX will only be used in Development enviroment

    **npm install tsx -D** 

*** Install ESLint 

    **npm i eslint @rocketseat/eslint-config -D**

    * After that you need ot install a VsCode extension ESLint
